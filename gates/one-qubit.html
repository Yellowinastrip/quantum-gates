Single-Qubit Gate Visualizer

Pick an input state from the unit circle, then toggle between Identity and NOT (Pauli‑X). (Normalization factors are omitted.)

Input states on the unit circle

Eight presets (±|0⟩ and ±|1⟩ and their ± superpositions) are placed evenly around the circle for an intuitive selection.

Gate

Identity (I)
NOT (X)

I leaves states unchanged. X swaps |0⟩ ↔ |1⟩.

How the NOT (X) gate acts

X|ψ⟩ with |ψ⟩ = a|0⟩ + b|1⟩ becomes a|1⟩ + b|0⟩. For the circle presets, a and b are ±1.

Built as a single HTML file with Bootstrap + Tailwind CDN. No build tools required.
